# 每日学习法

这份学习包的核心不是“多看”，而是每天形成可面试的输出。

## 每天固定流程

### 1. 八股答案卡片：60-90 分钟

每天 5 个题。

每个题按这个结构写：

- 一句话定义。
- 为什么需要它。
- 核心公式或流程。
- 优点和局限。
- 项目里怎么用。
- 面试官可能怎么追问。

### 三色打标

每道题每天只打一个标：

| 标记 | 含义 | 第二天怎么处理 |
| --- | --- | --- |
| 绿 | 30 秒和 2 分钟都能讲清，能接追问 | 三天后复述 |
| 黄 | 概念能说，但公式、shape、指标或项目例子不稳 | 第二天优先复习 |
| 红 | 说不成体系，或一追问就断 | 当天回完整学习章重学 |

每天只允许最多 5 道黄/红题进入第二天计划，避免错题库变成新的资料堆。

判断是否过关：

- 30 秒能说清定义。
- 2 分钟能讲完整机制。
- 被追问到公式、shape、复杂度、工程取舍时不慌。

### 101-140 补充追问怎么用

`03_高频题单100题.md` 末尾的 101-140 不是每天全刷，而是用在两种场景：

- 当天完整学习章读完后，用对应追问题测深度。
- 模拟面试前，从 101-140 随机抽 5 题做压力测试。

如果答题只会说名词，不会讲排查链路、评估指标或项目例子，就回到 [05_高频题单100题_答案索引.md](05_高频题单100题_答案索引.md) 找对应完整学习章。

### 2. 手撕代码：45-60 分钟

每天 1-2 题。

推荐顺序：

1. Self-Attention
2. Multi-Head Attention
3. RoPE
4. RMSNorm / LayerNorm
5. BPE
6. Top-k / Top-p
7. Beam Search
8. KV Cache
9. LoRA
10. DPO loss

每题记录：

- 输入输出 shape。
- 核心步骤。
- 最容易写错的地方。
- 时间复杂度和空间复杂度。
- 如果面试官要求不用 PyTorch 高阶 API，怎么简化。

### 3. 项目追问：30 分钟

每天挑一个项目问题回答。

高频追问：

- 你为什么这么做？
- 有哪些替代方案？
- 指标提升多少？
- 失败 case 是什么？
- 数据怎么来、怎么清洗？
- 线上怎么评估？
- 延迟和成本怎么控制？
- 如果重做会改什么？

### 4. 面经反推：20-30 分钟

每天看 3-5 篇 2025-2026 面经。

只记录三类题：

- 重复出现的题。
- 自己不会的题。
- 和自己简历项目相关的题。

看公司/岗位差异时，配合 [09_公司面经雷达与JD关键词.md](09_公司面经雷达与JD关键词.md) 打标签，不要把所有面经都当成同一类题。

不要记录：

- 明显过时题。
- 纯情绪帖子。
- 只卖资料包的标题。

### 5. 晚间复盘：10 分钟

问自己：

- 今天哪道题会被追问穿？
- 哪个代码题离开参考写不出来？
- 哪个项目回答太空？
- 明天只补哪三个点？

## 一道八股的合格答案

例题：KV Cache 是什么？

不合格：

> KV Cache 就是缓存 K 和 V，加快推理。

合格：

> KV Cache 是自回归解码时缓存历史 token 的 Key/Value，避免每生成一个新 token 都重新计算所有历史 token 的 K/V。Prefill 阶段会一次性计算 prompt 的 K/V，decode 阶段每步只计算新 token 的 Q/K/V，并用新 Q 和历史 K/V 做 attention。它能显著降低解码阶段重复计算，但会增加显存占用，显存大致随 batch、层数、头数、序列长度、head_dim 和 dtype 增长。常见追问包括如何估算 KV Cache 显存、MQA/GQA 怎么降低 KV Cache、长上下文时怎么管理缓存。

## 一道手撕题的合格记录

例题：MHA

记录：

- 输入 `x: [B, T, C]`
- 线性层得到 `qkv: [B, T, 3C]`
- reshape 成 `[B, T, 3, H, D]`
- 拆成 `q/k/v: [B, H, T, D]`
- attention score: `[B, H, T, T]`
- mask 后 softmax
- 输出 `[B, H, T, D] -> [B, T, C]`

易错：

- mask 维度 broadcast 错。
- transpose 后没有 contiguous，view 报错。
- 缩放因子应该是 `sqrt(head_dim)`。
- causal mask 方向写反。

## 一周节奏模板

周一到周五：

- 上午或晚上：八股 5 题。
- 手撕 1 题。
- 项目追问 1 个。

周六：

- 汇总本周错题。
- 重写本周最差的 3 道手撕。
- 做一次 60 分钟模拟面试。

周日：

- 看面经和岗位 JD。
- 调整下周计划。
- 休息半天，避免资料疲劳。
